# Node_Backend_Server

Node_Backend_Server is a backend server developed for react and next. It is designed for blog websites.


## Setup

Use NPM to install its depedencies.

```bash
npm install
```

Initiate the database using

```bash
npm run config
```

You can run the server using

```bash
npm start
```
